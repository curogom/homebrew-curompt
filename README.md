# Homebrew Tap for curo-prompt

Install `curo-prompt` using Homebrew:

```bash
brew tap curogom/curo-prompt
brew install curo-prompt
```

## Update

```bash
brew upgrade curo-prompt
```

## What is curo-prompt?

CLI tool for analyzing, evaluating, and optimizing LLM prompts.
